## Description

[Nest](https://github.com/nestjs/nest) framework with LDAP Passport authentication example.
<br>
Created by Simon Brännström.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start
```
```
# watch mode
$ yarn run start:dev
```
